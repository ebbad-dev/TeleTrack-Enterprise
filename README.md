# TeleTrack Enterprise

TeleTrack Enterprise is a network monitoring and incident management system designed to help organizations manage devices, facilities, technicians, alerts, incidents, and uptime performance in a structured way.

The system focuses on reliable network operations by tracking device status, recording incidents, assigning technicians, monitoring SLA performance, and maintaining audit logs for accountability.

## Features

* Device and facility management
* Technician assignment and tracking
* Incident and fault reporting
* Alert management
* SLA-based monitoring
* Audit logs for system activity
* Network uptime and performance reporting
* Structured database with optimized queries
* SQL procedures, triggers, and indexing

## Tech Stack

* Frontend: React, Vite, Tailwind CSS
* Backend: Flask, SQLAlchemy
* Database: SQLite / PostgreSQL
* Authentication: JWT
* Real-time Updates: Socket.IO
* Charts and Reports: Recharts
* Animations: Framer Motion

## Purpose

TeleTrack was developed as a database-focused enterprise project to demonstrate how real-world network monitoring systems manage faults, devices, technicians, alerts, and performance data.
